# 🎭 Anon-Market
Анонимная P2P-платформа для обмена монетами ANON. Работает прямо в консоли.

## 🌟 Возможности
- Полная анонимность ваших операций.
- Работает на любом устройстве с Python 3 (Termux, Linux, Windows).
- Легкий и быстрый код без лишних библиотек.

## 🚀 Быстрый старт
Скопируй и вставь эту команду в свой терминал:

```bash
curl -sL [https://raw.githubusercontent.com/zefcyka5-eng/Anon-marker/main/anon_market.py](https://raw.githubusercontent.com/zefcyka5-eng/Anon-marker/main/anon_market.py) > anon_market.py && python3 anon_market.py
